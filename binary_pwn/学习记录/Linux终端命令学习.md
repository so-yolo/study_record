#### touch

修改文件或者目录的时间属性，包括存取时间和更改时间。若文件不存在，系统会建立一个新的文件。

touch test.txt

touch ./father/son //在father的文件夹下建立文件son

#### vim

编辑和建立文本

vim test.txt

#### rm

rm test.txt //删除文件

rm -r * //删除当前目录下的所有文件及目录

#### chmod

修改文件权限

**![[Pasted image 20240923214805.png]]**

chmod 777 file

#### mkdir

建立文件夹

mkdir ./father/son //在father的文件夹下建立文件夹son
